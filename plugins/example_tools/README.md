# 示例工具包 (example_tools)

AutoFlow 官方示例插件，展示插件开发的完整流程。

## 提供的功能块

| 功能块 | 说明 |
|--------|------|
| 🎲 随机整数 | 在指定范围内生成随机整数，保存到变量 |
| ✂️ 文本处理 | 对文本进行转大/小写、反转、去空格、求长度操作 |
| 🌐 HTTP GET JSON | 发送 GET 请求并解析 JSON 字段，保存到变量 |

## 安装

此插件随 AutoFlow 附带，无需单独安装。

## 开发参考

请参阅 [插件开发文档](https://e.gitee.com/XinyuCraft-XYHC/repos/XinyuCraft-XYHC/autoflow-windows/blob/master/docs%2Fplugin-dev-guide.md) 了解如何开发自己的插件。
